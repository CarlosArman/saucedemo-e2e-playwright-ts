## [2.0.0] - 2026-04-19

### Added
- enterprise-ready project structure
- Playwright + Cucumber + TypeScript automation foundation
- Page Object Model architecture
- successful and failed login coverage
- shopping cart and basic checkout coverage
- multi-environment execution support (`DEV`, `UAT`, `PROD`)
- multi-browser execution support (`chromium`, `firefox`, `webkit`)
- sequential multi-browser execution strategy
- concurrent multi-browser execution strategy
- Allure report generation and opening commands
- Cucumber reporting outputs (HTML / JSON / JUnit)
- screenshot on failure support
- optional screenshot after each step
- Playwright trace support for failure analysis
- structured logging utilities
- documentation split into README + docs guides

### Improved
- README structure for recruiter-friendly and portfolio-oriented presentation
- bilingual documentation strategy (English / Español)
- commands, environment, and architecture documentation separation
